# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML and CSS.  
This project showcases skills, projects, and contact information with a clean UI, smooth animations, glassmorphism effects, and dark mode support.

---

## 🚀 Features

- Modern responsive design
- Smooth scrolling animations
- Glassmorphism UI effects
- Dark mode support
- Skills showcase section
- Projects showcase section
- Contact section with social links
- Mobile-friendly layout
- Clean and professional design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Web Design
- Flexbox
- CSS Grid
- Media Queries
